✅ 2. Dengan Generic

Penjelasan: Box<T> menggunakan generic, tipe data ditentukan saat instansiasi → AMAN.

📝 Kode Program:

class Box<T> {
  T data;
  Box(this.data);
}

void main() {
  var intBox = Box<int>(42);
  var stringBox = Box<String>("Hello");
  
  print("Nilai int: ${intBox.data}");
  print("Nilai string: ${stringBox.data}");
  // intBox.data = "Hello"; // ❌ ERROR compile-time!
}

💻 3. Sintaks Generic Class dan Function

class MyClass<T> {
  T data;
  MyClass(this.data);
  T getData() => data;
}

T myFunc<T>(T value) => value;

class Pair<K, V> {
  K key;
  V value;
  Pair(this.key, this.value);
  void printPair() => print("Key: $key, Value: $value");
}

void main() {
  var myInt = MyClass<int>(10);
  var myString = MyClass<String>("test");
  print(myInt.getData());
  print(myString.getData());
  
  var pair = Pair<String, int>("umur", 20);
  pair.printPair();
}

📦 4. Kelas Box<T>

class Box<T> {
  T? _value;
  Box(this._value);
  T? get value => _value;
  void setValue(T value) => _value = value;
}

void main() {
  var intBox = Box<int>(10);
  print("Int Box: ${intBox.value}");
  intBox.setValue(20);
  print("Set Int: ${intBox.value}");
  
  var stringBox = Box<String>("Hello");
  print("String Box: ${stringBox.value}");
  stringBox.setValue("World");
  print("Set String: ${stringBox.value}");
}

🔑 5. Multiple Type Parameters

🔗 6. Menggabungkan Dua Tipe Data Berbeda

String combine<T, U>(T value1, U value2) {
  return '$value1, $value2';
}

void main() {
  var result1 = combine<String, int>("Number is", 10);
  print(result1);
  
  var result2 = combine<int, double>(10, 20.5);
  print(result2);
}

🛡️ 7. Generic Dengan Extends

Penjelasan: <T extends num> membatasi tipe data hanya turunan dari num.

class NumberBox<T extends num> {
  T value;
  NumberBox(this.value);
  T getValue() => value;
}

void main() {
  var intBox = NumberBox<int>(10);
  print("Int: ${intBox.getValue()}");
  
  var doubleBox = NumberBox<double>(3.14);
  print("Double: ${doubleBox.getValue()}");
}

🔒 9. Pembatasan Tipe pada Interface

void printIterable<T extends Iterable<dynamic>>(T iterable) {
  for (var item in iterable) print(item);
}

void main() {
  printIterable<List<int>>([1, 2, 3]);
  printIterable<Set<String>>({'apple', 'banana'});
}

🐾 10. Pembatasan Tipe dengan Abstract Class

abstract class Animal { void makeSound(); }

class Dog implements Animal {
  void makeSound() => print("Woof!");
}

class AnimalCage<T extends Animal> {
  T animal;
  AnimalCage(this.animal);
  void sound() => animal.makeSound();
}

💼 11. Praktik Generic Class & Fungsi

class Repository<T> {
  final List<T> _items = [];
  void save(T item) => _items.add(item);
  List<T> getAll() => _items;
}

void swap<T>(List<T> list, int i, int j) {
  T temp = list[i]; list[i] = list[j]; list[j] = temp;
}

class Triple<A, B, C> {
  A first; B second; C third;
  Triple(this.first, this.second, this.third);
}

⚖️ 12. Generic vs Dynamic vs Object

Aspect Generic<T> dynamic Object
Type Safety ✅ Compile-time ❌ Runtime ✅ Compile-time
Performance ✅ Optimal ❌ Overhead ✅ Optimal
Casting Tidak perlu Tidak perlu Perlu casting 
Box<int> box1 = Box<int>(42);  // Generic: aman
dynamic box2 = 42;                // Dynamic: tidak aman
Object box3 = 42;                 // Object: perlu casting

🗄️ 13. Studi Kasus - Cache Generic

class Cache<T> {
  final Map<String, T> _storage = {};
  void set(String key, T value) => _storage[key] = value;
  T? get(String key) => _storage[key];
}

class Product { String name; int price; }

void main() {
  var cache = Cache<String>();
  cache.set('name', 'Alice');
}

📝 14. Latihan

class Stack<T> {
  final List<T> _items = [];
  void push(T item) => _items.add(item);
  T pop() => _items.removeLast();
  T peek() => _items.last;
}

List<T> mergeLists<T>(List<T> a, List<T> b) => [...a, ...b];

class Response<T> {
  T? data; String? error;
  Response.success(this.data) : error = null;
  Response.error(this.error) : data = null;
}

✨ Ringkasan Manfaat Generic

  • Type Safety - Error terdeteksi saat compile-time
  • Code Reusability - Satu class untuk berbagai tipe
  • No Casting - Tidak perlu konversi tipe manual
  • Performance - Lebih optimal dari dynamic
  • Constraints - Bisa batasi tipe dengan extends

📌 Kesimpulan Futuristik

  • ✅ Generic membuat kode TYPE-SAFE dan REUSABLE
  • ✅ Bisa digunakan di class, fungsi, dan collections
  • ✅ Type inference memudahkan penggunaan
  • ✅ Constraints (extends) membatasi tipe yang diperbolehkan
