1. Tanpa Generic

  • Pada struktur ini, sebuah kontainer atau variabel didefinisikan secara umum untuk menangani berbagai macam objek sekaligus.
  • Mekanisme: Variabel tersebut tidak dikunci pada satu tipe data spesifik, sehingga sebuah objek angka bisa dengan mudah digantikan oleh objek teks dalam alur yang sama.
  • Risiko: Karena sistem tidak memverifikasi kesesuaian tipe data saat kode ditulis, beban pemeriksaan berpindah sepenuhnya ke saat program dijalankan.
  • Dampak: Hal ini sering memicu munculnya exception atau kegagalan sistem secara mendadak.
  • Singkatnya, tanpa Generic, kode menjadi sangat dinamis namun rentan terhadap kesalahan tersembunyi.

2. Dengan Generic

  • Dalam skema ini, kontainer didefinisikan menggunakan parameter tipe formal yang bertindak sebagai cetak biru yang presisi.
  • Mekanisme: Saat objek dibuat, kita menentukan tipe data spesifik yang akan dikunci ke dalam wadah tersebut.
  • Keamanan: Sistem melakukan verifikasi ketat pada saat penulisan kode (compile-time).
  • Dampak: Menghilangkan kebutuhan konversi tipe data manual dan mengurangi risiko runtime error.

📝 Kode Program VS Code:

class Box<T> {
  T data;
  Box(this.data);
}

void main() {
  var intBox = Box<int>(42);
  var stringBox = Box<String>("Hello");
  
  print("Nilai int: ${intBox.data}");
  print("Nilai string: ${stringBox.data}");
}

3. Sintaks Generic Class dan Function

class MyClass<T> {
  T data;
  MyClass(this.data);
  T getData() => data;
}

T myFunc<T>(T value) => value;

class Pair<K, V> {
  K key; V value;
  Pair(this.key, this.value);
  void printPair() => print("Key: $key, Value: $value");
}

4. Kelas Box<T>

class Box<T> {
  T? _value;
  Box(this._value);
  T? get value => _value;
  void setValue(T value) => _value = value;
}

5. Multiple Type Parameters

class Pair<K, V> {
  K key; V value;
  Pair(this.key, this.value);
  void printPair() => print("Key: $key, Value: $value");
}

6. Menggabungkan Dua Tipe Data

String combine<T, U>(T value1, U value2) => '$value1, $value2';

void main() {
  print(combine<String, int>("Number is", 10));
  print(combine<int, double>(10, 20.5));
}

7. Generic Dengan Extends

<T extends num> membatasi tipe data hanya turunan dari num (int, double).

class NumberBox<T extends num> {
  T value;
  NumberBox(this.value);
  T getValue() => value;
}

8. Type Inference

T myFunc<T>(T value) => value;

void main() {
  var a = myFunc(10);     // int
  var b = myFunc("test"); // String
  print(a.runtimeType);
  print(b.runtimeType);
}

9. Pembatasan Tipe pada Interface

void printIterable<T extends Iterable<dynamic>>(T items) {
  for (var item in items) print(item);
}

10. Abstract Class Constraint

abstract class Animal { void makeSound(); }
class Dog implements Animal { void makeSound() => print("Woof!"); }
class AnimalCage<T extends Animal> {
  T animal;
  AnimalCage(this.animal);
  void sound() => animal.makeSound();
}

11. Praktik Generic Class & Fungsi

class Repository<T> {
  final List<T> _items = [];
  void save(T item) => _items.add(item);
  List<T> getAll() => _items;
}

void swap<T>(List<T> list, int i, int j) {
  T temp = list[i]; list[i] = list[j]; list[j] = temp;
}

class Triple<A, B, C> { A first; B second; C third; }

13. Cache Generic

class Cache<T> {
  final Map<String, T> _storage = {};
  void set(String key, T value) => _storage[key] = value;
  T? get(String key) => _storage[key];
}

14. Latihan

class Stack<T> {
  final List<T> _items = [];
  void push(T item) => _items.add(item);
  T pop() => _items.removeLast();
  T peek() => _items.last;
}

List<T> mergeLists<T>(List<T> a, List<T> b) => [...a, ...b];

class Response<T> { T? data; String? error;
  Response.success(this.data) : error = null;
  Response.error(this.error) : data = null;
}

🍂 KESIMPULAN

  • Universalitas Kode: Generic memungkinkan kelas dan fungsi beroperasi secara dinamis namun tetap terkontrol.
  • Keamanan Tanpa Kompromi: Bounded generic (extends) membatasi cakupan tipe data.
  • Struktur Data Terorganisir: Repository, Stack, Cache dapat digunakan tanpa menduplikasi kode.
  • Optimalisasi Fungsi: Fungsi generik seperti swap/combine melayani berbagai jenis data.
  • Intisari: Generic = kode reusable, type-safe, dan adaptif.
