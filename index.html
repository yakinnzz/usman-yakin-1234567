📦 1. Tanpa Generic

Penjelasan: Pada program ini, Box merupakan class yang digunakan untuk menyimpan data. Class Box menggunakan tipe data dynamic, sehingga dapat menampung berbagai jenis data. Namun karena tidak ada pembatasan tipe data, program berpotensi mengalami runtime error.

📝 Kode Program:

// Nama  : Usman Yakin Trio Sinada
// NIM   : 251410006
// Kelas : SI.2A

class Box {
  dynamic data;
  Box(this.data);
}

void main() {
  var box = Box(42);
  int value = box.data;
  print("Nilai awal: $value");
  
  box.data = "Hello";
  print("Data setelah diubah: ${box.data}");
}

💻 Output:

Output Program:
Nilai awal: 42
Data setelah diubah: Hello

⚠️ Masalah: Tidak ada pengecekan tipe saat compile-time, berpotensi runtime error!

✅ 2. Dengan Generic

Penjelasan: Box<T> digunakan sebagai class generic yang dapat menyimpan berbagai tipe data. Tipe <T> membuat class Box menjadi fleksibel namun tetap aman karena tipe data ditentukan saat instansiasi.

📝 Kode Program:

class Box<T> {
  T data;
  Box(this.data);
}

void main() {
  var intBox = Box<int>(42);
  var stringBox = Box<String>("Hello");
  
  print("Nilai int: ${intBox.data}");
  print("Nilai string: ${stringBox.data}");
  // intBox.data = "Hello"; // ❌ ERROR compile-time!
}

💻 Output:

Output Program:
Nilai int: 42
Nilai string: Hello

✅ Keuntungan: Type safety di COMPILE-TIME!

📖 3. Sintaks Generic Class dan Function

class MyClass<T> {
  T data;
  MyClass(this.data);
  T getData() => data;
}

T myFunc<T>(T value) => value;

class Pair<K, V> {
  K key;
  V value;
  Pair(this.key, this.value);
  void printPair() => print("Key: $key, Value: $value");
}

void main() {
  var myInt = MyClass<int>(10);
  var myString = MyClass<String>("test");
  print(myInt.getData());
  print(myString.getData());
  
  var pair = Pair<String, int>("umur", 20);
  pair.printPair();
}

📦 4. Kelas Box<T>

class Box<T> {
  T? _value;
  Box(this._value);
  T? get value => _value;
  void setValue(T value) => _value = value;
}

void main() {
  var intBox = Box<int>(10);
  print("Int Box: ${intBox.value}");
  intBox.setValue(20);
  print("Set Int: ${intBox.value}");
  
  var stringBox = Box<String>("Hello");
  print("String Box: ${stringBox.value}");
  stringBox.setValue("World");
  print("Set String: ${stringBox.value}");
}

🔗 6. Menggabungkan Dua Tipe Data Berbeda

  var result2 = combine<int, double>(10, 20.5);

}

🛡️ 7. Generic Dengan Extends

Penjelasan: <T extends num> membatasi tipe data hanya turunan dari num (int, double).

class NumberBox<T extends num> {
  T value;
  NumberBox(this.value);
  T getValue() => value;
}

void main() {
  var intBox = NumberBox<int>(10);
  print("Int: ${intBox.getValue()}");
  
  var doubleBox = NumberBox<double>(3.14);
  print("Double: ${doubleBox.getValue()}");
  
  // var stringBox = NumberBox("test"); // ❌ Error!
}

🎯 8. Type Inference (Inferensi Tipe)

T myFunc<T>(T value) => value;

void main() {
  var intValue = myFunc(10); // Dart inferensi T sebagai int
  print(intValue.runtimeType); // Output: int
  
  var stringValue = myFunc("test"); // Dart inferensi T sebagai String
  print(stringValue.runtimeType); // Output: String
}

🐾 10. Pembatasan Tipe dengan Abstract Class

abstract class Animal {
  void makeSound();
}

class Dog implements Animal {
  void makeSound() => print("Woof!");
}

class Cat implements Animal {
  void makeSound() => print("Meow!");
}

class AnimalCage<T extends Animal> {
  T animal;
  AnimalCage(this.animal);
  void sound() => animal.makeSound();
}

void main() {
  AnimalCage<Dog>(Dog()).sound();
  AnimalCage<Cat>(Cat()).sound();
}

💼 11. Praktik Generic Class & Fungsi

// Repository Generic
class Repository<T> {
  final List<T> _items = [];
  void save(T item) => _items.add(item);
  T? getById(int id) => id < _items.length ? _items[id] : null;
  List<T> getAll() => _items;
}

// Fungsi swap
void swap<T>(List<T> list, int i, int j) {
  T temp = list[i];
  list[i] = list[j];
  list[j] = temp;
}

// Class Triple
class Triple<A, B, C> {
  A first; B second; C third;
  Triple(this.first, this.second, this.third);
}

⚖️ 12. Generic vs Dynamic vs Object

Aspect Generic<T> dynamic Object
Type Safety ✅ Compile-time ❌ Runtime ✅ Compile-time
Flexibility Terbatas (sesuai T) Sangat fleksibel Terbatas
Performance ✅ Optimal ❌ Overhead ✅ Optimal
Casting Tidak perlu Tidak perlu Perlu casting

dynamic box2 = 42;

🗄️ 13. Studi Kasus - Cache Generic

class Cache<T> {
  final Map<String, T> _storage = {};
  
  void set(String key, T value) => _storage[key] = value;
  T? get(String key) => _storage[key];
  void remove(String key) => _storage.remove(key);
}

class Product {
  String name;
  int price;
  Product(this.name, this.price);
  @override
  String toString() => 'Product($name, \$$price)';
}

void main() {
  var userCache = Cache<String>();
  userCache.set('name', 'Alice');
  
  var productCache = Cache<Product>();
  productCache.set('p1', Product('Laptop', 1000));
}

📝 14. Latihan

// 1. Stack
class Stack<T> {
  final List<T> _items = [];
  void push(T item) => _items.add(item);
  T? pop() => _items.isEmpty ? null : _items.removeLast();
  T? peek() => _items.isEmpty ? null : _items.last;
}

// 2. Merge Lists
List<T> mergeLists<T>(List<T> a, List<T> b) => [...a, ...b];

// 3. Response
class Response<T> {
  T? data;
  String? error;
  Response.success(this.data) : error = null;
  Response.error(this.error) : data = null;
}

✨ Ringkasan Manfaat Generic

  • Type Safety - Kesalahan tipe data terdeteksi saat compile-time
  • Code Reusability - Satu class/function bisa digunakan untuk berbagai tipe
  • No Casting Needed - Tidak perlu konversi tipe data manual
  • Performance - Lebih optimal dibanding dynamic
  • Constraints - Bisa membatasi tipe yang diperbolehkan dengan extends

📌 Kesimpulan

  • ✅ Konsep Generic pada Dart memungkinkan class dan function bekerja dengan berbagai tipe data
  • ✅ Generic membuat program menjadi lebih fleksibel karena satu kode dapat digunakan untuk banyak tipe data
  • ✅ Penerapan bounded generic seperti extends membantu membatasi tipe data agar lebih aman
  • ✅ Generic class seperti Repository, Stack, Cache, dan Pair memudahkan pengelolaan data secara terstruktur
  • ✅ Generic function seperti swap dan combine meningkatkan efisiensi tanpa perlu membuat banyak fungsi berbeda
  • ✅ Penggunaan Generic dalam OOP membuat program lebih reusable, aman, dan mudah dikembangkan
